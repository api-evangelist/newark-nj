# Newark, NJThis is a repo for managing machine readable artifacts for the [Newark, NJ APIs](http://data.ci.newark.nj.us/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Newark, NJ APIs](http://data.ci.newark.nj.us/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://data.ci.newark.nj.us/)- [Developer](http://data.ci.newark.nj.us/)The [APIs.json](https://github.com/api-evangelist/newark-nj/blob/master/apis.json) index provides links to all the relevant links for the Newark, NJ APIs, including the OpenAPI and Postman if they are present.
